## Idea

A **cone over a [cospan](foundation.cospans.md)** `A -f-> X <-g- B` with domain
`C` is a triple `(p, q, H)` consisting of a map `p : C → A`, a map `q : C → B`,
and a homotopy `H` witnessing that the square

```text
      q
  C -----> B
  |        |
 p|        |g
  V        V
  A -----> X
      f
```

commutes.
